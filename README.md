# Simulador de Investimentos

## Descrição do Projeto

Em um cenário de altas taxas de juros, uma alternativa ao financiamento é aguardar e investir o dinheiro antes de adquirir um bem. Essa estratégia permite evitar juros e, muitas vezes, aumentar o poder de compra ao longo do tempo.

Este projeto consiste em um aplicativo que simula investimentos mensais, permitindo ao usuário visualizar quanto seu dinheiro pode render ao longo do tempo.

O usuário informa:
- Valor investido mensalmente  
- Número de meses  
- Taxa de juros mensal  

E o sistema retorna:
- Montante acumulado sem juros  
- Montante acumulado com juros compostos  

---

## Tecnologias Utilizadas

- Flutter  
- Dart  
- Material Design  

---

## Como Executar o Projeto

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/simulador-investimentos.git
```

2. Acesse a pasta do projeto:
```bash
cd simulador-investimentos
```

3. Instale as dependências:
```bash
flutter pub get
```

4. Execute o aplicativo:
```bash
flutter run
```

---

## Funcionamento do Aplicativo

1. Insira o valor que deseja investir mensalmente  
2. Informe o número de meses  
3. Adicione a taxa de juros mensal  
4. O sistema calculará automaticamente:
   - Valor total investido (sem juros)  
   - Montante final com juros compostos
